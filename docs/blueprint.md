# **App Name**: LocalBasket

## Core Features:

- Product Listings: Allow local stores to list their products with descriptions, images, and prices.
- Browse Stores: Enable users to browse available stores and their products.
- Shopping Cart: Implement a shopping cart feature for users to add and manage items.
- Order Placement: Allow users to place orders and specify delivery details.
- Order Management: Enable store owners to view and manage incoming orders. 
- AI Product Suggestions: Provide AI-driven product recommendations based on past purchases or items in the user's cart; use an LLM tool to decide when and how the suggested products should appear to users, in order to provide maximum user convenience. 
- Store Locator: Users will provide their location and then the app lists nearby available stores.

## Style Guidelines:

- Primary color: Soft Green (#90EE90) to evoke freshness and nature.
- Background color: Very light green (#F0FFF0), near-white, for a clean backdrop.
- Accent color: Light orange (#FFB347) for calls to action, like adding items to the cart.
- Font: 'PT Sans' sans-serif for headlines and body, offering readability with a touch of warmth.
- Use clear, simple icons for product categories and navigation.
- A clean, grid-based layout to showcase products effectively.
- Subtle transitions and animations for a smooth user experience.
